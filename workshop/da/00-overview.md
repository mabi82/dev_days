# 🎮 VS Code GitHub Copilot Agent Lab

[🎮 Spil spillet](https://dotnet-presentations.github.io/vscode-github-copilot-agent-lab/) • [📚 Lab Guide](https://dotnet-presentations.github.io/vscode-github-copilot-agent-lab/docs/) • [← README](../../README.da.md)

---

> **Varighed:** ~1 time
> **Niveau:** Intermediate
> **Stack:** C# / .NET 10 / Blazor WebAssembly

Velkommen til Soc Ops - en hands-on workshop, hvor du forvandler en simpel Social Bingo-app til noget helt særligt med VS Code Agent Mode og GitHub Copilot.

---

## 📋 Hurtig tjekliste

Før du starter, tjek:

- [ ] VS Code **v1.107+** (ingen pending updates)
- [ ] Logget ind med **GitHub Copilot** (Pro, Business eller Enterprise)
- [ ] Git installeret
- [ ] .NET 10 SDK installeret
- [ ] Chat-panel åbent og Agent klar

> 💡 **Tip:** Brug DevContainer for et prækonfigureret miljø.

---

## 🎯 Hvad du lærer

| # | Færdighed | Beskrivelse |
|---|------------|-------------|
| 1 | **Context Engineering** | Lær AI om din kodebase med instruktioner |
| 2 | **Agentic Primitives** | Brug background agents, cloud agents og custom workflows |
| 3 | **Design-First Development** | Lad AI iterere UI mens du styrer retningen |
| 4 | **Test-Driven Development** | Brug TDD agents til pålidelig feature-udvikling |
